    https://aws.plainenglish.io/deploying-a-docker-container-in-aws-using-fargate-5a19a140b018

    Prerequistes, need the following installed:

    • Docker
    • AWS CLI

    Create Dockerfile for App

    • Create Dockerfile

    "Deploy" Docker Image to ECR

    In ECR:

    • Create a repository
    • Set to Private
    • Give it a name
    • Create
    • This gives you a URI of the format: [some numbers].dkr.ecr.[region].amazonaws.com/[name]

    Push Docker image to ECR:

    • Build Docker image
    • Tag Docker image with the ECR repository: docker tag my-web-app:latest .dkr.ecr..amazonaws.com/my-web-app:latest
    • Authenticate Docker with ECR: aws ecr get-login-password --region | docker login --username AWS --password-stdin .dkr.ecr..amazonaws.com
    • If worked should see a "Login Succeeded" message
    • Push Docker image to ECR: docker push .dkr.ecr..amazonaws.com/my-web-app:latest
