    AWS Associate Developer Exam Details

    2024/09/21
    Tags: AWS AWS Certification AWS Associate Developer

    Introduction

    I covered the content of the AWS Associate Developer exam in a previous post, this post is just to cover some of the exam details.

    Details

    • Costs 150 USD
    • 130 minutes in length
    • Passing score is 720/1000
    • 65 questions, includuing 15 unscored
    • Scenario-based questions
    • Multiple choice and multiple response answers
    • Qualification valid for 3 years
